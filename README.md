# Lekho (লেখো) — Bangla Phonetic Keyboard for macOS

Lekho brings Avro Phonetic-style typing back to the modern Macs.

**[Download](https://github.com/ARahim3/Lekho/releases/latest)** | **[Website](https://arahim3.github.io/Lekho/)**

---

## What is this?

If you've used Avro Keyboard on Windows or iAvro on older macOS or OpenBangla in Linux, you know how natural phonetic Bangla typing can feel. But every existing option for macOS is either abandoned or broken or not very "Avro-style" on Apple Silicon.

Lekho fills that gap. It's a native macOS input method that works in every app — Safari, Chrome, VS Code, Notes, Spotlight, everywhere.

## Features

- **Avro Phonetic-style Typing** — type `ami banglay gan gai` → আমি বাংলায় গান গাই
- **150k word dictionary** with smart suggestions and autocorrect
- **Native Apple Silicon** — ~2.6 MB download, instant startup, zero CPU when idle
- **Works everywhere** — built with Apple's InputMethodKit framework
- **Completely offline** — no internet needed, no data collection, no telemetry
- **Free** — no ads, no subscription

## Install

1. Download the latest `.dmg` from [Releases](https://github.com/ARahim3/Lekho/releases/latest)
2. Open the DMG and double-click **Install Lekho.pkg**
3. Log out and log back in (first time only — macOS needs this to discover new input methods)
4. Go to **System Settings → Keyboard → Input Sources → Edit**, click **+**, find **Lekho**, and add it
5. Use Globe key or Ctrl+Space to switch

## Requirements

- macOS 13 (Ventura) or later
- Apple Silicon Mac (M1/M2/M3/M4/M5)

## Credits

Lekho is powered by [OpenBangla's riti engine](https://github.com/OpenBangla/riti) — the same transliteration engine behind [OpenBangla Keyboard](https://github.com/OpenBangla/OpenBangla-Keyboard) on Linux.
